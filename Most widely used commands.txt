BASIC commands to be used in Vs code:
 
For MacOs,
command + A -- select the entire cell
command + C -- copy the content
command + V -- paste the copied text
command + S -- save all the changes

 rnpc   - reactNativePureComponent
 rnpce  - reactNativePureComponentExport
 rcc    - reactClassComponent

 rce    - reactClassExportComponent
 rcep   - reactClassExportComponentWithPropTypes
 rconst - classConstructor
